# Проектная работа в колледже

Это моя **первая проектная работа**, выполненная во время учёбы в колледже по специальности "Веб-разработчик".

Проект представляет собой Лендинг.

---

**Цель работы:** Закрепить навыки вёрстки, работы с PHP и базами данных.

## 🛠️ Стек технологий

Проект разработан с использованием следующего стека:

- **Языки разметки и стилей:** HTML5, CSS3
- **Фреймворк:** Bootstrap 5.0.2 (для адаптивной сетки и компонентов)
- **Язык программирования:** JavaScript (ES6)
- **Шрифты:** Google Fonts (Poppins)
- **Дизайн:** Figma (макет для вёрстки)
- **Структура:** Модульная файловая структура с разделением на HTML, CSS, JS и изображения
